# Jumpnrun Prototype

Prototype for side-view platform movement, jump physics, pickups, and simple hazards.

The scene uses:

- Sunnyland Foxy/Opossum/items/props/background assets
- ASCII terrain with direct Sunnyland tile texture mapping
- explicit script-side movement and gravity
- hardcoded ground/platform heights for collision

Map collision generation is not implemented yet, so the physics script mirrors the visible map layout manually for now.
